📊 Process Yield Analysis (FTY, RTY)
📌 Sobre o projeto

Este projeto simula um processo produtivo com múltiplas etapas e calcula métricas de rendimento da qualidade, utilizadas em melhoria de processos e gestão da qualidade.

Foram implementados cálculos de:

FTY (First Time Yield)

RTY (Rolled Throughput Yield)

RTY Total

Taxa de refugo por processo

Fluxo de entrada, saída e defeitos

Esses indicadores são amplamente utilizados em metodologias como Six Sigma e Lean Manufacturing para avaliar a eficiência real de processos produtivos.

⚙️ Tecnologias utilizadas

Python

NumPy

Pandas

Random (simulação de variabilidade de processo)

🧠 Conceitos aplicados
FTY — First Time Yield

Probabilidade de uma unidade passar por uma etapa do processo sem retrabalho ou defeitos.

RTY — Rolled Throughput Yield

Probabilidade de uma unidade atravessar todo o processo sem defeitos.

RTY=FTY1​×FTY2​×FTY3​…FTYn

Esse indicador mostra o impacto acumulado das perdas ao longo do processo.


🎯 Objetivo do projeto

Demonstrar, de forma prática, como métricas de qualidade podem ser aplicadas para:

Avaliação de eficiência de processos

Identificação de perdas ocultas

Análise de desempenho produtivo
